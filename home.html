URL Shortener Microservice

Example creation usage:

https://hrly.herokuapp.com/new/https://www.google.com
https://hrly.herokuapp.com/new/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://hrly.herokuapp.com/8170" }

Usage:

https://hrly.herokuapp.com/1

Will redirect to:

https://www.google.com/
